Installing Software
 - Installing the FRC Game Tools
 - Uninstall Old Versions
 - Download the FRC Game Tools from NI.
	https://www.ni.com/en-us/support/downloads/drivers/download.frc-game-tools.html#369633
 - Get NI license / serial number from FIRST website --> dashboard --> Payment & Product --> Passwords/Voucher Codes --> NI

 - Follow the github link, and select your OS from "Assets" at the bottom
 - Follow instructions and open VS Code from either desktop shortcut or Windows search.
